# Laboratorio: Gestión de Tareas con Servlet

Este es un proyecto web desarrollado con Java, Servlets y JSP para la gestión de una lista de tareas en memoria, aplicando el patrón PRG (Post/Redirect/Get).

## Requisitos para ejecutar
- Java JDK 17 o superior
- Apache Tomcat 10.x u 11.x
- Maven 3.8+

## Instrucciones de Ejecución
1. Clonar este repositorio en tu computadora.
2. Abrir el proyecto usando IntelliJ IDEA.
3. Ir a **Run -> Edit Configurations** y configurar un servidor local de **Tomcat**.
4. En la pestaña de **Deployment**, agregar el artefacto `gestion.tareas:war exploded`.
5. Ejecutar el servidor (Play).
6. Abrir el navegador en la ruta: `http://localhost:8080/gestion-tareas/tareas`
